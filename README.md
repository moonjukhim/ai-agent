##### 환경설정

1. 적절한 위치에 코드 저장

```bash
mkdir 02
cd 02
```

2. Virtual Env 생성

```bash
python -m venv .venv
```

3. venv 활성화

```bash
.venv\Scripts\activate
# deactivate
```

4. 패키지 업데이트

```bash
python -m pip install --upgrade pip
```

5. 종속성 설치 후 주피터랩 기동

```bash
pip install jupyter langchain langchain-openai jupyter jupyterlab
jupyter lab
# http://localhost:8888/lab
```
